# Search Papers — 使用指南

集成 arXiv · CrossRef · Scopus · PubMed 四大数据库，DeepSeek AI 全流程辅助阅读。

---

## 快速开始

```shell
cd search_papers
pip install -r requirements.txt
cp .env.example .env        # 填入 DEEPSEEK_API_KEY
python app.py
```

---

## 核心工作流

**1. 搜索** — 输入研究主题，AI 自动优化为英文查询

**2. AI 分析** — 搜索后自动提示，生成结构化报告：
- 推荐论文 Top 5（中英文标题）
- 分类论文列表（按子方向分组）
- 研究趋势 & 进一步建议

**3. 细读单篇** — `detail <编号>`：查看详情 + AI 深度解读 + 图形摘要下载

---

## 常用命令

| 命令 | 说明 |
|:---|:---|
| `search <主题>` | 搜索研究主题 |
| `depth <1-5>` | 搜索深度（1=一层，2+=追踪引用链） |
| `analyze` | 对当前结果 AI 分析 |
| `detail <编号>` | 细读单篇（含专属阅读指南） |
| `ask <问题>` | 就当前主题自由提问 |
| `guide` | 查看通用阅读框架 |
| `export [文件名]` | 导出搜索结果 Markdown |
| `save [文件名]` | 保存 AI 分析报告 |
| `savelog [文件名]` | 保存终端完整记录 |
| `help` | 显示所有命令 |

---

## 阅读指南解读

每篇 `detail` 后，AI 给出**专属阅读指南**：

1. **本文填补的研究空白** — 前人没做什么，本文做了什么
2. **推荐阅读顺序** — 先读哪个部分及原因
3. **核心图表** — 先看哪张图/哪个实验
4. **潜在局限性** — 基于领域知识的客观批评
5. **一句话标签** — `[年份]作者_核心结论_缺陷`
6. **找同类文献** — 祖师爷（奠基工作）和后浪（后续改进）

输入 `guide` 可查看通用阅读框架：**10秒定去留，5分钟抓骨架**。

---

## 链接策略

所有链接优先使用 **DOI 直链**（`doi.org/10.xxxx`），校园网可直接访问出版商页面。
无 DOI 的论文使用 **Google Scholar 标题搜索** 作为保底。

---

## 快捷技巧

- 直接输入主题（不加命令）→ 等同于 `search`
- `depth` 不带参数 → 查看当前深度
- `list` → 重新显示论文列表
