# LLM Wiki

This directory is the integrated research Wiki used by Search Papers.

- `CLAUDE.md` defines the LLM maintenance rules.
- `raw/` stores original research assets and is intentionally excluded from git except for directory placeholders.
- `sources/literature/` stores paper notes generated by `wiki_bridge.py` and completed by an LLM agent.
- `entities/`, `concepts/`, `synthesis/`, and `explorations/` store the evolving knowledge graph.

Use Search Papers `ingest <N>` to create a metadata stub, then hand the generated agent prompt to an LLM to complete the Wiki workflow.
